# travelPlannerApp
App to plan out travel itineraries



## Usage
---
Rename .env.sample to .env and add requires environment variables. Using port 3000 for frontend and 4000 for backend

### Install dependencies
```
#Backend deps
npm install

#Frontend deps
cd frontend
npm install
```

### Run server
```
npm run server
```

### Run app
```
npm run dev
```
